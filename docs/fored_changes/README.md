# Forked Changes

This directory tracks fork-only changes that are intentionally maintained in
the monorepo after the local fork histories were rewritten and consolidated.

Current forked repositories with maintained changes:

- `pypto`
- `simpler`
- `pto-isa`

Per-commit documentation lives under:

- `docs/fored_changes/pypto/`
- `docs/fored_changes/simpler/`
- `docs/fored_changes/pto-isa/`

Each file records:

- the rewritten commit id
- the reason the change exists
- the files touched
- the verification used to confirm the change is still required
